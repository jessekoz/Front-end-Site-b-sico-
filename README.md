# 💻 Site Básico Front-End — Projeto de Estudo

Este é um projeto básico de site desenvolvido com **HTML5** e **CSS3**, com o objetivo de praticar e consolidar os fundamentos do desenvolvimento Front-End.

O projeto simula uma página informativa sobre tecnologias web, contendo layout com header, seção principal, cards de conteúdo e botões de ação.

---

## 📚 Objetivo

- Praticar estruturação semântica com HTML
- Trabalhar estilização com CSS
- Criar layout com seções e artigos
- Aplicar organização de conteúdo em cards
- Simular página real de apresentação

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

---

## 🧱 Estrutura da Página

O site contém:

- ✅ Header com menu de navegação
- ✅ Banner principal com imagem de destaque
- ✅ Seções informativas
- ✅ Cards sobre:
  - HTML
  - CSS
  - JavaScript
- ✅ Botões "Saiba mais"
- ✅ Layout centralizado
